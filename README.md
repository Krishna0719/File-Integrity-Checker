## ‚öôÔ∏è Features

The File Integrity Checker offers a variety of features to help you secure your files:

- **RSA Digital Signatures**: Ensures that files are verified by a trusted source and have not been altered.
- **X.509 Certificates**: Provides a framework for secure connections and trusted files.
- **Cryptographic Hashing**: Uses MD5 and SHA-256 hashing algorithms to monitor files' integrity.
- **AES Encryption**: Protects sensitive files with strong encryption.
- **Tamper Detection**: Alerts you in case of unauthorized changes or modifications.
- **Audit Logging**: Keeps a detailed record of all actions taken by the application for your reference.

## üîç How It Works

1. **File Monitoring**: The application continuously checks the integrity of specified files. It calculates hashes and compares them over time.
2. **Alerts**: If a file's hash does not match the original, you will receive an alert to take action.
3. **Logging**: All actions taken on the files are logged for transparency. You can view this log any time to understand which files were checked or altered.
